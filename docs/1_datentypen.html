There are important differences between dataframes and matrices. Note especially what Df[4] returns compared to Mat[4]:

Key difference: For a matrix, Mat[4] returns the 4th element column-wise. For a dataframe, Df[4] returns the 4th column. Always use Df[2,3] or Df.iloc[2,3] for single elements.

Dataframes can be merged using merge():

merge() by default performs an Inner Join – only rows with matching keys in both dataframes appear in the result. Use all=TRUE to keep all rows (Outer Join).
